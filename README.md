# 🤖 Telegram AI Chatbot (n8n)

An AI-powered Telegram chatbot built using **n8n**, **Groq**, and **Telegram Bot API**.

## 🚀 Features

- AI-powered responses using Groq LLM
- Telegram integration
- Conversation memory
- Calculator tool support
- Modular n8n workflow
- Easy to import and customize

## 🛠️ Tech Stack

- n8n -> Workflow automation 
- Groq API ->  AI language model
- Telegram Bot API ->  Messaging platform
- AI Agent ->  Response generation
- Buffer Memory ->  Conversation history

## 📂 Files

- `telegram-ai-chatbot-workflow.json` – Main n8n workflow

## 📥 How to Use

1. Import the workflow into n8n.
2. Create your own Telegram credentials.
3. Create your own Groq credentials.
4. Connect the credentials.
5. Activate the workflow.
